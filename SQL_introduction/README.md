# SQL Introduction

This project contains SQL scripts that demonstrate basic database operations and queries.

## Requirements

- MySQL 8.0 or higher
- Ubuntu 20.04 LTS

## Files

- `0-list_databases.sql` - Lists all databases in MySQL server

## Usage
```bash
cat 0-list_databases.sql | mysql -hlocalhost -uroot -p
```
