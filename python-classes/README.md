# Python - Classes and Objects

## Description
This project covers the basics of Object-Oriented Programming (OOP) in Python, focusing on classes, objects, attributes, and methods.

## Learning Objectives
* What is OOP
* What is a class
* What is an object and an instance
* What is the difference between a class and an object or instance
* What is an attribute
* How to use public, protected and private attributes
* What is `self`
* What is `__init__` method
* What is Data Abstraction, Data Encapsulation, and Information Hiding

## Requirements
* Python 3.x
* All files are interpreted/compiled on Ubuntu 20.04 LTS
* All files must be executable
* Code should use the pycodestyle style (version 2.7.*)
